📈 How Admin Login Works

Visit Admin Login
Go to admin.html page where the login form is displayed.
2
Enter Credentials

3
Submit Form
Click the "Login to Admin Portal" button to verify credentials.
4
System Verification
JavaScript checks if credentials match (admin/admin123).
5
Show Loading
Button changes to "🔄 Redirecting..." to show it's processing.
6
Redirect Dashboard
After 1 second, redirects to admin-dashboard.html with full access.

🔒 Security Features

🔐

Password Protection

All passwords are entered in password fields and masked. Admin users should use strong, unique passwords in production.

✅

Form Validation

Login forms require both username/email and password fields to be filled before submission is allowed.

🔄

Session Management

Each user type (Admin, Teacher, Student) has separate login pages and dashboards with different access levels.

📱

Mobile Responsive

All login pages are fully responsive and work on mobile phones, tablets, and desktop computers.

🚪

Logout Feature

Each dashboard has a logout button that returns users back to their login page with session cleared.

👤

Profile Display

Dashboard shows logged-in user's name and role to ensure they know what account is active.
